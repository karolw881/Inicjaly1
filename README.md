# Inicjaly1

Celem Projektu było zaprogramowanie wlasnych inicjalow za pomoca krzywych Bezier. Projekt w całości wykonany jest w javie. Bibliotekami użytymi w programie są 
java.awt.* , javax.swing.JPanel. Stworzone zostały klasy Main , Surface , Obraz, Bezier. Klasa Main jest odpowiedzialna za wystartowanie klasy Obraz. 
Klasa Surface jest odpowiedzialna za stworzenie powierchni inicjału. Powstają w niej  obiekty klasy Bezier z odpowiednimi parametrami. Klasa Bezier jest algorytmem 
służącym do stworzenia danej krzywej Bezier. W konstruktorze tej klasy zapełniamy dwie tablice punktów x i y. Nastęnie w funkcji paint , w pętli z parametrem t
rysujemy daną krzywą.
